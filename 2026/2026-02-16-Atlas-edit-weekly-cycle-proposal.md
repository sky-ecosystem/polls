# Atlas Edit Weekly Cycle Proposal - February 16, 2026

The Core Facilitators have placed an [Atlas Edit Weekly Cycle Proposal](https://sky-atlas.io/#A.1.10.2.2) into the [voting system](https://vote.sky.money/polling) [on behalf of Ranked Delegate Blue](https://forum.sky.money/t/atlas-edit-weekly-cycle-proposal-week-of-2026-02-16/27720/4). This Governance [Poll](https://sky-atlas.io/#A.1.10.2.2) will be active for three days beginning on Monday, February 16 at 16:00 UTC.

**This is a binary vote.**

- **You may vote for a single option.**
- **You should vote for the option which you prefer.**
- **If you would accept either option, you should vote 'Abstain'.**

## Review

The community may vote in this poll to express support or opposition to the following Atlas Edit Weekly Cycle Proposal:

- [Atlas Edit Pull Request](https://github.com/sky-ecosystem/next-gen-atlas/pull/186)
- [Proposal Discussion Thread](https://forum.sky.money/t/atlas-edit-weekly-cycle-proposal-week-of-2026-02-16/27720)

A brief summary of this Atlas Edit has been provided by the Author and is shown below:

_This proposal includes the following edits:_

- _**Revise Aligned Delegate Requirements** - Adds additional responsibilities for Aligned Delegates and defines a new response framework for handling breaches._
- _**Authorize Safe Harbor Agreement** - Authorizes Sky to enter into the Safe Harbor Agreement to allow whitehat hackers to return recovered funds to the Protocol._
- _**Update Risk Parameters For Anchorage Digital** - Updates the short-term parameters for Anchorage Digital in the Risk Framework._
- _**Clarify References To Ozone** - Clarifies that Ozone is the Operational Executor Agent and Soter Labs is Operational GovOps._
- _**Update Names Of Obex Foundation And Development Company** - Updates the names of the Obex Foundation and Development Company to Rubicon and Treadstone, respectively._
- _**Add Integration Boost Instances To Skybase Artifact** - Adds Integration Boost Instances to the Skybase Artifact for Curve and Morpho._
- _**Update Grove Artifact For Upcoming Spell** - Authorizes planned deployments by Grove in the upcoming Executive Spell._
- _**Correct Nesting in stUSDS BEAM Documents** - Corrects the nesting of documents related to the stUSDS BEAM to more accurately reflect the logical hierarchy._
- _**Correct Needed Research Documents** - Corrects the order of paragraphs in two Needed Research documents._
- _**Add Launch Agent 7 Artifact** - Adds a new Artifact for Launch Agent 7._

## Outcomes

This poll implements a **Minimum Positive Participation** value. The Minimum Positive Participation for Atlas Edit Weekly Cycle Proposals is currently set to **480,000,000 SKY**.

**If the votes for the 'Yes' option exceed the votes for the 'No' option AND the votes for the 'Yes' option equal or exceed 480,000,000 SKY, then the following actions will be taken:**

- The associated Pull Request will be merged into The Atlas.
